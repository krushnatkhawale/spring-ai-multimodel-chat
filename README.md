# spring-ai-multimodel-chat

This is a Spring AI Multimodel Chat application.

## Getting Started

To run this project, you will need to have Java and Maven/Gradle installed.

### Prerequisites

* Java 17 or higher
* Maven 3.6.3 or higher (if using Maven)
* Gradle 7.x or higher (if using Gradle)

### Building and Running

You can build and run the application using your preferred build tool.

#### Gradle

```bash
./gradlew bootRun
```

#### Maven

```bash
./mvnw spring-boot:run
```
